Admin course card component Displays course metadata, term information, location, and offers navigation for managing additional details or electronic resources. Features: - Shows department, instructors, and term info. - Checks if course term is active based on the current date. - Provides button to view/edit course details. - Provides button to manage electronic resources (if set).
